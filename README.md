

Tech Stack

The following tools and libraries are used in this project:
-Python: Programming language (version 3.x).
-Jupyter Notebook: Interactive environment for running the code.
-NumPy: Library for numerical computations and handling arrays.
-Pandas: Library for data manipulation and analysis.
-Kaggle API: For downloading datasets from Kaggle.
-Google Colab: Cloud-based environment for running the notebook (with optional GPU support).
